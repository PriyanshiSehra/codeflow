# CodeFlow - Sorting Algorithm Visualizer

An interactive web application that visualizes popular sorting algorithms through animated bars, helping users understand how different algorithms work step by step.

## Features

- Interactive sorting visualizations
- Multiple sorting algorithms
- Adjustable array generation
- Smooth animations
- Responsive interface

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

## Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

## Getting Started

Clone the repository:

```bash
git clone https://github.com/PriyanshiSehra/codeflow.git
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Visit:

```
http://localhost:3000
```
